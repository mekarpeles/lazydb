LazyDB

LazyDB is a basic wrapper around the Python shelve flatfile dbm module.

Why Awesome

LazyDB can serialize/pickle arbitrary python objects and write them to a flatfile db. Creating a database is as easy as instantiating an object. LazyDB provides a familiar python interface to that of s3 and couchdb.

Installation

$ pip install lazydb

Example

#!/usr/bin/env Python

from lazydb import Db

class LazyUser:
      def __init__(self, name):
          self.name = name

db = Db('/home/user/db')
db.put('user', LazyUser('lazybones'))
u = db.get('user')
u.name
